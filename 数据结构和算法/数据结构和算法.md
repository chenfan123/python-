### 时间复杂度

时间复杂度 T

主要条件：随着问题规模而变化的
次要条件：随着问题规模变化而不变的

On^2：主要条件是 n^2，次要条件是 1
T(n) = O(n^2)

#### 时间复杂度计算规则

1. 基本操作 O(1)
2. 顺序结构 T(n) = O(n)
3. 循环结构，复杂度按乘法计算
4. 分支结构，时间复杂度取最大值
5. 判断一个算法的效率时，往往只需要关注操作数量的最高次项，其他次要项和常数项可以忽略
6. 在没有特殊说明时，我们所分析的算法的时间复杂度都是指最坏时间复杂度

常见的时间复杂度
O(1) < O(logn) < O(n) < O(nlogn) < O(n^2) < O(n^3) < O(2^n) < O(n!) < O(n^n)

### 空间复杂度

在运算中临时占用存储空间大小的度量 S(n)

O(1)：普通常量、变量、对象、元素数量与输入数据大小 N 无关的集合，皆使用常数大小的空间。
O(n)：元素数量与 N 呈线性关系的任意类型集合（常见一维数组、链表等），均使用线性大小的空间。
O(n^2)：元素数量与 N 呈平方关系的任意类型集合，均使用平方大小的空间。
