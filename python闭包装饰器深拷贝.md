# 闭包装饰器和深拷贝

## 闭包

作用：保存函数内的变量，而不会随着调用完函数而被销毁。

什么是闭包：**使用外部函数变量**的**内部函数**称为闭包

构成条件：

1. 有嵌套；（在函数嵌套的前提下）
2. 有引用：内部函数使用了外部函数的变量（还包括外部函数的参数）
3. 有返回：外部函数返回了内部函数名

```python
# 闭包
def test():
    a = 100

    def test_in(b):
        print(a + b)
    return test_in


f = test()
f(6)
```

### nonlocal 关键字

global：声明全局变量
nonlocal：声明能够让内部函数去修改外部函数的变量

## 装饰器

注意：一个装饰器的参数有且只有一个
作用：不改变原有函数的基础上，给原有函数增加额外功能，装饰器本质上就是一个闭包函数。
构成条件：

1. 有嵌套；（在函数嵌套的前提下）
2. 有引用：内部函数使用了外部函数的变量（还包括外部函数的参数）
3. 有返回：外部函数返回了内部函数名
4. 有额外功能：给需要装饰的原有函数增加额外功能

##### 格式一：传统写法

```python
装饰后的函数名 = 装饰器名（被装饰的原函数名）
装饰后的函数名()
```

##### 格式 2:语法糖

```python
@装饰器名
def 函数名():
    pass
```

##### 多个装饰器的使用

离函数最近的装饰器先装饰，然后外面的装饰器再进行装饰，由内到外的装饰过程。如果用装饰器的语法糖来写，看到的效果是从上往下执行

```python
@装饰器名1
@装饰器名2
def 函数名():
    pass
```

##### 带参数装饰器

使用带参数的装饰器其实就是在装饰器外面又包裹来一个函数，使用该函数接受参数，返回装饰器
