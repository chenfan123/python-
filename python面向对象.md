# python 提高 - 面向对象

## 面向对象

1. 面向过程：把所需要的步骤列出来，按照步骤写代码挨个实现。强调的是以**步骤**为基础完成各种操作。
2. 面向对象：是一种编程思想，强调的是以**对象**为基础完成各种操作，基于面向过程的。说到面向对象，不得不提的三大特性：封装、继承、多态。

##### 三大特性

###### 封装

隐藏对象的属性和实现细节，仅提供对外的方法让别人去访问

###### 继承

子类继承父类的属性和方法，使得子类对象具有父类的特征和行为

###### 多态

不同类的对象对同一消息作出相应，即同一个函数在不同场景下表现出不同形态。

### 类

#### 类基础

类是对象的抽象，是属性和行为的集合；对象是类的具体实例。

定义方法必须要写 self 参数，self 代表当前对象

```python
class Car:
    # 属性
    name = '宝马'
    price = 1000000
    # 行为
    def run(self):
        print(f'{self.name}在行驶')

    def stop(self):
        print(f'{self.name}在停止')
```

##### self

是 python 内置的关键字，用于指向对象实例本身。

#### 添加和获取对象属性

- 类外面添加和获取对象属性：`对象名.属性 = 属性值`，`对象名.属性`

- 类内添加和获取对象属性：添加需要结合\_\_init\_\_方法来实现，`self.属性 = 属性值`，获取直接`self.属性`

> 累内如何设置属性，要结合\_\_init\_\_方法来实现。

#### 魔法方法

是 python 的内置函数,在特定的场景会被调用

##### \_\_init\_\_方法

当新创建一个对象时，就会自动触发。

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
```

##### \_\_str\_\_方法

当用 print()函数打印对象的时候，会自动调用该对象（所在类）的 str 魔法方法。该魔法方法默认打印的是对象的地址值，改为打印对象的各个属性值。

```python
class Car:
    def __init__(self, name, price):
        self.name = name
        self.price = price

    def __str__(self):
        return f'打印的是这个{self.name}'
```

##### \_\_del\_\_方法

当删除对象时（调用 del 删除对象或文件执行结束后）,python 解释器会默认调用

```python
# __del__ 方法
class Dog:
    def __init__(self, name):
        self.name = name

    def __del__(self):
        print(f'{self.name} 对象被删除了')

    def __str__(self):
        return f'打印的是这个{self.name}'

dog = Dog('旺财')
del dog
```

### 继承

什么是继承？ - 子类继承父类的属性和方法，使得子类对象具有父类的特征和行为`class 子类名(父类名):`

#### 单继承和多继承

##### 单继承

单继承就是一个子类只能继承自一个父类，不能继承多个类。

##### 多继承

多继承就是一个子类可以继承自多个父类，可以继承多个类的属性和方法。
当一个类有多个父类是，默认使用第一个父类的同名属性和方法，可以使用`类名.__oro__`方法来查看调用顺序,即先找哪个类，后找哪个类

##### 子类重写父类同名方法和属性

当子类属性火方法与父类的属性或方法名字相同时，子类可以重写父类的属性和方法。

##### 子类调用父类方法

子类中仍想要保留父类的行为，则需要在子类中调用父类方法，可以直接使用父类名来调用，使用的方法：`父类名.父类方法名(self)`
两种方式：

1. super().父类方法名() - 调用父类方法
2. 父类名.父类方法名(self) - 调用父类方法
   第一种精准访问，想找哪个父类就调哪个父类；第二种只能访问最近的那个父类，有就用，没有就报错
